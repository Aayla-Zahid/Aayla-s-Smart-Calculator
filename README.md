# 🧮 Aayla's Smart Calculator

A modern, keyboard-friendly calculator built with **Python** and **Streamlit**. This calculator provides a clean and interactive interface for performing both basic arithmetic and advanced mathematical operations in real time.

## ✨ Features

- ➕ Basic arithmetic operations
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
  - Modulus (%)
  - Exponentiation (**)

- 🔢 Scientific functions
  - Sine (sin)
  - Cosine (cos)
  - Tangent (tan)
  - Logarithm (log₁₀)
  - Factorial (x!)
  - Pi (π)
  - Euler's Number (e)
  - Reciprocal (1/x)

- ⌨️ Keyboard-controlled input
- 📝 Live expression input
- 🧹 Clear (C) button
- ⌫ Backspace support
- 🔄 Automatically closes missing parentheses
- ⚡ Instant calculation
- 🛡️ Safe expression evaluation using a restricted execution environment
- 🎨 Simple, clean, and responsive Streamlit interface
